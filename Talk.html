Quick introduction to copulas’ place within fin. metrics

  • Within financial econometrics we use a vast array of models to capture co-movements in the asset markets
  • Theoretical problem arises when we talk about dependence
    • Linear correlation is only appropriate when the dependence structure (or joint distribution) follow a normal distribution

Sklar’s theorem

Definition (Copula):

\[C(u_1, \ldots, u_d) = P(U_1 \leq u_1, \ldots, U_d \leq u_d), (u_1, \ldots, u_d) \in (0,1)^d\]

Thus Sklar’s theorem states:

\[C(F_1(x_1), \ldots,F_d(x_d)) = P(U_1 \leq F_1(x_1), \ldots, U_d \leq F_d(x_d))\] \[ = P(F_1^{-1}(U_1) \leq x_1, \ldots, F_d^{-1}(U_d) \leq x_d)\] \[ = F(x_1, \ldots, x_d)\]
